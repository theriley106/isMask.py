# IsMask.py
Realtime Face Mask Detection in ~10 lines of Python Code

## Instructions

### CLI

``` {.sourceCode .bash}
$ python isMask.py
FACE MASK OFF
FACE MASK OFF
FACE MASK OFF
FACE MASK ON
FACE MASK ON
...
```

### Web App

``` {.sourceCode .bash}
$ ./run.sh
[INFO] Listening at: http://127.0.0.1:8000 (970)
```

And visit `http://127.0.0.1:8000` in your browser.
